知耳知币说明

知币是专供知耳用户在知耳平台使用的虚拟货币，通过发布内容，参与知耳官方活动获取，知币可用于知耳平台的试用产品的兑换，以及消耗于部分专属福利活动。除此之外，不得用于其他用途。

知币是知耳平台奖励用户发布内容的运营福利，不得转让他人，不得用于交易，用作其他牟利手段。知币也无法用于提现。知耳拥有知币的解释权与回收权。

用户违反知耳平台的规定或是国家法律相关条款，知耳有权对该账号的知币进行回收，和没收该账号的再次获取知币的权利。

知币获取

  • - 首次登录可以获得5个知币；
  • - 连续登录（点击个人签到），每日可获取1个知币，连续5天额外获得5个知币；
  • - 首次完善全部的个人资料可获得5个知币；
  • - 首次完善全部的账号资料可获得2个知币；
  • - 首次填写肤质可获得2个知币；
  • - 首次给知耳好评可获得5个知币；
  • - 首次点赞其他用户的作品可获得1个知币；
  • - 首次评论其他用户的作品可获得1个知币；
  • - 首次回复其他用户的评论可获得1个知币；
  • - 首次收藏其他用户的作品可获得1个知币；
  • - 发布PO图1篇可获得1个知币，每日发布的前20篇PO图可获得，当天超过20次PO图不再获取知币；
  • - 发布日记1篇可获得2个知币，每日发布的前10篇日记可获得，当天超过10篇后不再获取知币。当日发5篇日记额外奖励5个知币；
  • - 发布视频1个可获得2个知币，每日发布的前10个视频可获得，天超过10篇后不再获取知币。当日发5篇日记额外奖励5个知币；
  • - 在问答板块回复问题的答案被置顶可获得5个知币；
  • - 被其他用户邀请回复可获得2个知币；
  • - 认证达人可获取20个知币；
